# Meme Autopsy

Meme Autopsy is an AI forensic engine for meme projects.

Instead of generating hype, it diagnoses why a meme narrative may go viral, decay, or die. The product treats internet-native assets like forensic cases, producing a structured autopsy report across narrative strength, community cohesion, symbolic durability, lore depth, and attention resilience.

## Why it exists

Most tools in the meme ecosystem focus on creation:
- generating content
- launching tokens
- tracking market activity

Very few tools explain why a meme project actually survives or fails.

Meme Autopsy was built to shift the category from generation to diagnosis.

## What it does

Users submit:
- a project name
- a narrative description
- an optional website URL
- optional tweets or community language

The system returns:
- a final verdict
- a root-cause analysis
- a six-factor forensic breakdown
- a collapse timeline
- intervention suggestions

## Features

- Forensic case-style UI with dark cinematic aesthetic
- Verdict classification (Dead on Arrival / Viral but Fragile / Stable Cult Potential / Decaying Fast)
- Narrative and community health scoring
- Collapse timeline analysis
- Suggested interventions
- Demo-ready sample cases (DogePriest, BananaFax, SaintMeme)
- Chinese / English language switching
- Responsive design

## Forensic dimensions

- Narrative Coherence
- Meme Spreadability
- Symbol Stickiness
- Community Trust
- Lore Depth
- Attention Resilience

## Tech stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Context for i18n

## Getting started

```bash
npm install
npm run dev
```

## Demo flow

1. Open the landing page
2. Load a sample case or enter a meme project
3. Run the autopsy
4. Review the verdict and root cause
5. Inspect the forensic breakdown
6. Explore the collapse timeline
7. Read intervention suggestions
